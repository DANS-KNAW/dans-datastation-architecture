SURF Data Archive¶

Data Archive is a tape storage facility provided by SURF, the collaborative ICT organization for Dutch education and research. It is used by the DANS Data Vault service to store its finished layers. The interface to Data Archive is provided by the dmftar utility, which is provided by SURF as open source software. See Data Vault Storage Root for more information about the DANS Data Vault layers.
