External interfaces¶

The following interfaces are exposed to the outside world:

  • The Dataverse user interface. This interface used by human users.
  • The Dataverse RESTful HTTP API. This interface is used by internal and external systems that need to interact with Dataverse.
  • A SWORD2 deposit service. This interface is used by external systems that want to deposit data in Dataverse or directly in the Data Vault.
  • Long term preservation copies of the data via the Data Vault Storage Root. This interface (or exchange format) is intended for future users who need to disseminate the data stored in the Data Vault after the current dissemination infrastructure has been decommissioned.
